## v5.9.6 (2026-06-01) — 리그 인원 부족 시 인접 리그 자동 병합

### 🐛 버그 수정

#### 4인 미만 에러로 대진 생성이 막히던 문제 해결
- **기존 증상**: 일부 리그 인원이 4명 미만이면 `st.stop()` 으로 전체 대진 생성 불가
- **변경**: 부족 리그를 막지 않고 인접 리그로 그날만 일시 흡수하여 편성
- 신규 헬퍼 `merge_insufficient_leagues()` 동작:
  - 부족 리그(4명 미만) → 바로 위 상위 리그로 흡수 (B부족→A, C부족→B)
  - 최상위 A가 부족하면 → 바로 아래 B로 내림
  - 흡수받은 리그가 다시 부족하면 연쇄 처리
  - `player_code` 리그 접두 1글자만 교체 (성별·이름·게스트 태그 보존)
  - 회원 선택 모드(`AM홍길동`)·자동 생성 모드(`AM01`) 모두 지원
- 병합 발생 시 안내 메시지 표시
- 병합 후에도 4명 미만이면 그때 안내 후 중단
- **원본 리그 소속·기록 분류는 그대로**, 오늘 생성된 대진에만 적용

### 📑 구조
- 진단 expander 문구 갱신: "4명 미만 오류 디버깅" → "리그 인원·병합 디버깅"
- 버전 v5.9.5 → v5.9.6

## v5.9 (2026-06-01) — 기록실 분리 및 개인기록실 신설

### ✨ 신규

#### 기록실 분리 — 통합기록실 / 개인기록실
- 사이드바 메뉴를 `🏆 기록실` 1개에서 `🏆 통합기록실` + `👤 개인기록실` 2개로 분리
- **통합기록실**: 기존과 동일하게 전체 선수 누적 통계 운영
- **개인기록실**: 회원 개인 전적 조회 전용 페이지 신설

#### 개인기록실 — 회원명 검색
- 회원명 입력 후 엔터로 즉시 조회, 유사 이름 버튼 제안, 전체 회원 목록 펼치기

#### 개인기록실 탭 1 — 월별 리그 타임라인
- 선택 연도의 12개월 전체 표시, 리그명 배지(컬러), 연도 선택기(최근 4년)

#### 개인기록실 탭 2 — 베스트페어 / 워스트페어
- 같은 팀으로 뛰었을 때 승률 높은/낮은 파트너 순위
- 월간/연간 선택, 최소 2경기 이상 집계, 베스트(초록)/워스트(빨강) 컬러 카드 UI

#### 개인기록실 탭 3 — 라이벌 전적
- 상대팀으로 뛴 선수별 맞대결 승무패, 월간/연간 선택
- 특정 선수 selectbox 필터, 전체 선택 시 요약 카드 + 승률 색상 3단계 구분

### 🐛 버그 수정

#### 라이벌 전적 요약 카드 이중 집계 수정
- **원인**: 복식 상대팀 2명에게 각각 1번씩 집계 후 합산 → 1경기 = 2카운트
- **수정**: 요약 카드용 `match_totals`를 경기(매치) 단위로 별도 집계

#### 휴면 시작일 도래 시 category 자동 전환 미동작
- **원인**: `category` 컬럼이 저장 버튼 클릭 시에만 갱신
- **수정**: `load_df()`에서 `dormant_period` 기반으로 `category` 메모리상 자동 보정
  - 진행 중 휴면 기간 있음 → `"휴면"` 적용 / 모든 기간 종료 → `"정회원"` 복귀
  - 탈퇴는 변경하지 않음, 시트 원본 건드리지 않음
